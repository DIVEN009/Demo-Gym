# Titan Fitness Gym - Demo Website

A modern, responsive single-page website for a gym business demo.

## Features

✅ **Fully Responsive** - Works perfectly on mobile, tablet, and desktop
✅ **Modern Design** - Dark theme with red accents, bold typography
✅ **Interactive Elements** - Smooth scrolling, hover effects, animations
✅ **Call-to-Action** - WhatsApp and Call buttons throughout
✅ **SEO Optimized** - Semantic HTML structure
✅ **Fast Loading** - Optimized CSS and JavaScript
✅ **Mobile-First** - Touch-friendly buttons and navigation

## Sections

1. **Hero Section** - Eye-catching headline with CTA buttons
2. **Why Choose Us** - Feature cards highlighting gym benefits
3. **Services** - Grid layout showcasing all gym services
4. **Membership Plans** - Monthly, Quarterly, and Yearly options
5. **Timings & Location** - Operating hours and Google Maps
6. **Trust Section** - Trainer profiles and testimonials
7. **Final CTA** - Strong call-to-action before footer

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- Google Fonts (Bebas Neue, Montserrat)

## Deployment

### Deploy to Vercel

1. Install Vercel CLI:
```bash
npm i -g vercel
```

2. Deploy:
```bash
cd Demo-Gym
vercel
```

### Or use Vercel Dashboard

1. Go to [vercel.com](https://vercel.com)
2. Import the `Demo-Gym` folder
3. Deploy with one click

## Customization

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-red: #e31e24;
    --dark-bg: #0a0a0a;
    --dark-gray: #1a1a1a;
}
```

### Update Contact Info
Replace placeholder phone number in `index.html`:
- Search for: `919876543210`
- Replace with your number (include country code)

### Modify Content
All content is in `index.html` - edit text, headings, and descriptions directly.
